# 👁️ OmniEye AI

AI-powered real-time suspicious activity monitoring and surveillance system using Computer Vision, Deep Learning, and FastAPI.

---

## 🚀 Features

* 🔍 Real-time suspicious activity detection
* 🔫 Weapon detection using YOLO models
* 🧠 AI-powered activity analysis
* 📹 Live MJPEG video streaming
* 🌐 WebSocket-based alert system
* ⚡ FastAPI backend for high-performance processing
* 🎯 Adaptive detection pipeline
* 🖼️ Clean overlay visualization system
* 📊 Health monitoring endpoints

---

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/harshdave05/OmniEye.git
cd OmniEye
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run Backend

```bash
python sentinel_backend.py
```
